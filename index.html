    Name 3 differences between Web1, Web2 and Web3:


    Web1
    1) No interaction.
    2) Users can only recieve information.

    Web2
    1) Interaction between users.
    2) Encourage users to share information.
    3) Web Applications.

    Web3
    1) Uses AI algorithms to learn about the user, and to discover new relevant content.
    2) Can iterpret information like humans.
    3) Distributes relevant information for human's needs.

    Example:
    A user that will search "Golden Retriver", in Web1 the user will get information pages,
    in Web2 the user will get Facebook page of Golden retriver Fans and will be allowd to "Like" this page and leave comments on posts.
    In Web3 the computer will offer the user assosiation of adopting dogs, which offers to adopt Golden retriver dog,
    based on his location, last searchs and his browsing history.
    The results based on Web3 will be different and more dynamic depends on indevidual users.
